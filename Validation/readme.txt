	==============================================
	SampleTemplate Steps
	==============================================
	1) Make copy of SamplesTemplate folder
	2) Rename copied folder to new Project Name
	3) Start Eclipse 
	4) 'Import Project' from copied folder
	5) Rename Package Name from dev.ronlemire.samplestemplate to new Package Name
	6) In AndroidManifest.xml rename <manifest package="dev.ronlemire.samplestemplate"> to new Package Name
	7) In strings.xml change <string name="app_name">ProjectTemplate to new Project Name
	8) Fix any package reference errors to R object from dev.ronlemire.samplestemplate.R to new Package Name
